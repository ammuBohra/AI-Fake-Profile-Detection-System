## AI Fake Instagram Profile Detection System
AI/ML based system for detecting fake and scam social media profiles using machine learning algorithms.

## Project Overview
This project is an AI-powered Fake Instagram Profile Detection System built using:

- Python
- Machine Learning
- Flask
- HTML/CSS
- Random Forest Algorithm

The system predicts whether an Instagram profile is Fake or Genuine based on profile features.


##  Features
- Fake profile detection
- Machine learning prediction
- Confidence score prediction
- Flask web application
- User-friendly interface
- Serialized ML model using Pickle
- Real-time prediction


## Machine Learning Algorithm

- Random Forest Classifier


